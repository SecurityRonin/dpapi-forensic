# Privacy Policy

*Last updated: 2026-06-24*

## Summary

dpapi-forensic is a local Rust library. It does not collect, transmit, or store any personal data on remote servers.

## Data Access

dpapi-forensic reads only the file bytes you pass to it. All processing — parsing and decryption — happens in memory on your local machine. Nothing is uploaded anywhere.

## Telemetry

dpapi-forensic has **no telemetry**. It makes no network requests of any kind.

## Open Source

dpapi-forensic is open source (Apache-2.0). You can audit every line of code at [github.com/SecurityRonin/dpapi-forensic](https://github.com/SecurityRonin/dpapi-forensic).
